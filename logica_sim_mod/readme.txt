About the sim
===============

The original simulator with some functionality changes.It is intended to be run in Eclipse IDE, but can easily run on any platform.

Some changes

-Correct delivery reporting
-Sending x amount of MO's with an specified interval between them.

All credits go to Logica on a fine simulator.


Running the simulator
======================

After the checkout in Eclipse simply right click on the project and run as java application.

The main class is Simulator.
